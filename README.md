# WSMD
Discovery motifs from ChIP-seq datasets with WSMD
=================================================

Depends:
-------
  * R ( ≥ 3.2.5 )
  * Matrix ( ≥ 1.2-6 )
  * Rcpp( ≥ 0.12.6 ）
  * Rmosek（≥ 7.1.2 ）

Input and output examples:
-------------------------
  * Inout sequence files: input_foreground_sequences.fasta, input_background_sequences.fasta
  * Output motif file: output_motif.txt
